# badge

Specifies a badge's value or glyph.

## Element hierarchy

**&lt;badge&gt;**

## Syntax

``` syntax
<badge value    = "1-99" | "none" | "activity" | "alert" | "alarm" | "available" | ...
       version? = integer />
```

### Key

`?`   optional (zero or one)

### Child Elements

None.

### Parent Elements

This outermost (document) element may not be contained by any other elements.
