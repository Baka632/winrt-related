# toast




Base toast element, which contains at least a single [**visual**](element-visual.md) element.

## Element hierarchy

**&lt;toast&gt;**

## Syntax

``` syntax
<toast launch?   = string
       duration? = "long" | "short" >

  <!-- Child elements -->
  visual,
  audio?,
  commands?

</toast>
```

### Key

`?`   optional (zero or one)
