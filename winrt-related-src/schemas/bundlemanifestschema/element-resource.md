## Syntax

``` syntax
<Resource Language?       = language
          Scale?          = "100" | "120" | "140" | "150" | "160" | ...
          DXFeatureLevel? = "dx9" | "dx10" | "dx11" />
```

### Key

`?`   optional (zero or one)
