# NetworkSettings (CarrierControlSchema_v2 schema)


Defines the network settings in a subscriber's carrier provisioning file.

## Syntax

``` syntax
<NetworkSettings>

  <!-- Child elements -->
  IPv4LinkMTU?,
  IPv6LinkMTU?,
  DNSRetrySettings?

</NetworkSettings>
```

### Key

`?`   optional (zero or one)
