# KeyValue


Defines a single public key as specified in [XML DSIG](https://www.w3.org/TR/xmldsig-core/).

## Syntax

``` syntax
<KeyValue>

  <!-- Child elements -->
  ( DSAKeyValue
  | RSAKeyValue
  | any element in a non-schema namespace{}
  )

  <!-- This element may also contain text (mixed content). -->

</KeyValue>
```

### Key

`{}`   specific range of occurrences
