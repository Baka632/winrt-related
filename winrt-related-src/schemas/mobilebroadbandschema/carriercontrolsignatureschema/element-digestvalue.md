# DigestValue


Defines the digest value as specified in [XML DSIG](https://www.w3.org/TR/xmldsig-core/). The algorithm used to generate [**DigestValue**](element-digestvalue.md) is defined in [**DigestMethod**](element-digestmethod.md).
