## Syntax

``` syntax
<DigestMethod Algorithm = anyURI >

  <!-- Child elements -->
  any element in a non-schema namespace*

  <!-- This element may also contain text (mixed content). -->

</DigestMethod>
```

### Key

`*`   optional (zero or more)
