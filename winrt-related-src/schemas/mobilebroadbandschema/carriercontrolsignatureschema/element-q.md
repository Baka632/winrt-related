# Q


Defines an integer in the range 2\*\*159 &lt; [**Q**](element-q.md) &lt; 2\*\*160 which is a prime divisor of [**P**](element-p.md)-1 as specified in [XML DSIG](https://www.w3.org/TR/xmldsig-core/).
