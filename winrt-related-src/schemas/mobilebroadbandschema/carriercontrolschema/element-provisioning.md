# Provisioning (CarrierControlSchema schema)


Defines parameters used to establish trust and refresh settings for future provisioning attempts.

``` syntax
<Provisioning>

  <!-- Child elements -->
  TrustedCertificates?,
  RefreshParameters?

```

### Key

`?`   optional (zero or one)
