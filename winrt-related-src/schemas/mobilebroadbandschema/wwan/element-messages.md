# Messages


Contains a set of MNO messages that are parsed by Windows 8 and may be signaled to the user.

## Element hierarchy

**&lt;Messages&gt;**

## Syntax

``` syntax
<Messages>

  <!-- Child elements -->
  Message+

</Messages>
```

### Key

`+`   required (one or more)

## Attributes and Elements


### Attributes
