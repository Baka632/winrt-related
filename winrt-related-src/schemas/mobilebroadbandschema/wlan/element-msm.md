# MSM


Defines various media-specific module (MSM) settings for this profile on a WLAN.

## Syntax

``` syntax
<MSM>

  <!-- Child elements -->
  security?,
  any element in a non-schema namespace*

</MSM>
```

### Key

`?`   optional (zero or one)
`*`   optional (zero or more)
