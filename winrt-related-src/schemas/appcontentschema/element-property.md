## Syntax

``` syntax
<Property Key   = string
          Type? = "bstr" | "boolean" | "string" | "wstr" | "astr" | ... : "string" >

  <!-- Child elements -->
  Value*

  <!-- This element may also contain text (mixed content). -->

</Property>
```

### Key

`?`   optional (zero or one)

`:`   default value

`*`   optional (zero or more)
