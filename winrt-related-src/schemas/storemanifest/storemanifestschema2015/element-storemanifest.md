# StoreManifest


Root node for the StoreManifest schema (for Windows 10).

## Element hierarchy

**&lt;StoreManifest&gt;**

## Syntax

``` syntax
<StoreManifest>

  <!-- Child elements -->
  ( DeviceCompanionApplication?
  & Dependencies?
  )

</StoreManifest>
```

### Key

`?`   optional (zero or one)
`&`   interleave connector (may occur in any order)
