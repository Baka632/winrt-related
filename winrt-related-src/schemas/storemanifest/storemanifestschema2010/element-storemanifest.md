# StoreManifest (Store manifest schema for Windows 8.1 and earlier)


Root node for the StoreManifest schema (for Windows 8.1 and earlier).

## Element hierarchy

**&lt;StoreManifest&gt;**

## Syntax

``` syntax
<StoreManifest>

  <!-- Child elements -->
  ProductFeatures?

</StoreManifest>
```

### Key

`?`   optional (zero or one)

## Attributes and Elements


### Attributes

None.
