# SupportedFileTypes (package schema for Windows 8, child of FileTypeAssociation)


Defines the file types associated with the app. They are unique per package and are case sensitive.

``` syntax

  <!-- Child elements -->
  FileType{1,1000}

</SupportedFileTypes>
```
