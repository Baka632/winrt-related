## Syntax

``` syntax
<Interface Name        = A string between 1 and 255 characters in length that consists of alpha-numeric, period, and underscore characters.
           InterfaceId = A GUID in the form xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx. />
```
