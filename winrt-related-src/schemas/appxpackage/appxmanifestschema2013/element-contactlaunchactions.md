# ContactLaunchActions




Declares actions to take when a contact is launched.

<dd>
<dl>

</dd>

## Syntax

``` syntax
<ContactLaunchActions DesiredView? = "default" | "useLess" | "useHalf" | "useMore" | "useMinimum" >

  <!-- Child elements -->
  LaunchAction{1,50}

</ContactLaunchActions>
```

### Key

`?`   optional (zero or one)
`{}`   specific range of occurrences
