# Extension

Declares an extensibility point for the app.

## Element hierarchy

**&lt;Extension&gt;**

## Syntax

``` syntax
<Extension Category = "windows.lockScreenCall" | "windows.contact" | "windows.appointmentsProvider" | "windows.alarm" >

  <!-- Child elements -->
  ( Contact
  | AppointmentsProvider
  )?

</Extension>
```

### Key

`?`   optional (zero or one)
