# uap:SupportedUsers (Windows 10)


Indicates whether or not the package is multi-user aware. This setting is used at install time to determine whether the package can be installed on the system.

## Syntax

``` syntax
<SupportedUsers>

  "single" | "multiple"

</uap:SupportedUsers>
```
