# uap:Url (Windows 10)


Specifies a URL to which a plugin may send cookies. Need only be a valid URI; not necessarily a URL.

## Syntax

``` syntax
<Url>

  A string between 1 and 32767 characters in length in the form of a valid web url.

</uap:Url>
```
