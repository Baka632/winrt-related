## Syntax
```syntax
<uap4:ContactPanel SupportsUnknownContacts = Boolean. >               
```

## Attributes
| Attribute | Description | Data type | Required |
|-----------|-------------|-----------|----------|
| SupportsUnknownContacts | True if the contact panel supports unknown contacts, otherwise false. | Boolean. | Yes |
