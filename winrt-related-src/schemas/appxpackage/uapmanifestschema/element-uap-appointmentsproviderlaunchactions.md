## Syntax

``` syntax
<AppointmentsProviderLaunchActions DesiredView? = "default" | "useLess" | "useHalf" | "useMore" | "useMinimum" >

  <!-- Child elements -->
  uap:LaunchAction{0,10}

</uap:AppointmentsProviderLaunchActions>
```

### Key

`?`   optional (zero or one)
`{}`   specific range of occurrences
